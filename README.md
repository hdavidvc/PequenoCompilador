# Pequeño compilador y generación de código en a c++ utilizando C# 

## Documentacion
```
Tarea Pequeño Compilador 
Este analizar reconoce las siguientes palabras:
1. inicio
2. proceso
3. fin
4. si
5. ver
6. mientras
7. entero
8. cadena
```
## Ejemplo
```c#


 inicio ;
cadena var1 # "Bienvenidos" ;

entero var5 # 1 ;

proceso ;

mientras ( var5 < 5 )

{

ver "Año correcto" ;

}


si ( var1 == "Bienvenidos" )

{

ver "COMPILADORES 2023" ;

}

fin ;

```
